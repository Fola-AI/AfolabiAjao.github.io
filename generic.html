VOLVO BI PROJECT

In this project, I analyzed a two-year Volvo Automobile dataset using Power BI to uncover key business insights. The interactive dashboard highlights top-performing models, identifies the most profitable vehicles, and showcases the best-performing dealerships. Key metrics such as total revenue, retail orders, and regional performance trends are visualized to support data-driven decision-making. This project demonstrates my ability to transform raw data into actionable insights through intuitive and dynamic visual storytelling.

CLIENT SERVICES GOODWILL PROJECT

This Power BI dashboard was created as a demonstration project for the Client Services team at a high-end automotive brand, inspired by real-world scenarios involving McLaren vehicles. The dataset used in this project is entirely synthetic (fake) and was generated solely for learning and portfolio purposes. It does not reflect actual customer or vehicle data and should not be interpreted as real. The dataset includes: Simulated case records including customer names, vehicle VINs, and unique case IDs Owner and agent details Goodwill Authorisations representing fictional gestures of compensation or service goodwill The dashboard showcases how Power BI can be used to: Track and visualise Goodwill requests and approvals across customer service cases Identify trends in case handling and agent involvement Provide interactive filtering by car model, case handler, or service gesture type This project demonstrates how Power BI can be leveraged to support customer experience insights, improve service strategy, and support operational decisionsâ€”even when working with mock data in a controlled environment. ="actions special">
